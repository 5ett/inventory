Guidelines

When adding a new user there are few things one has to take note of in order to have a completley smooth experience.

  • Make sure the name is properly typed in; both first and last.
  • The user will have to choose personal password after authorization so a simple password will suffice for now.
  • Make sure you type in a correct email in order to ensure confirmation link is sent to added user
